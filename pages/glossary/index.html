Glossary of term that I use on this site

This is collection of the definitions of words that I use on this site and might be unfamiliar to the readers.

Hugo

This is a piece of software that allows me to generate this site from a couple of text based files and configurations.

It generates the pages of this site as static pages. That means that the pages are stored in files that are ready to be used by your browser. When a page is requested the server does nothing other than reading the file and sending it over to your computer. This makes it very, very fast and efficient. The only down-side is that a little bit of work needs to be done when setting up the site and I need to re-generate it when I add content. But it’s super fast. If you are a computer geek, you can read more about it straight from the source: gohugo.io

SSH

Secure SHell: a way of accessing a server (usually Linux based one) for administration purposes - such as running commands, changing configurations or accessing files. It uses end-to-end encryption for the data traffic, that’s why it’s called “secure”. And “shell” refers to a text based command line interface, where one types in commands to get things done.
